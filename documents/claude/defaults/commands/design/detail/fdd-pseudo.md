# FDD Pseudo-code Generation

## Purpose
Transform FDD document into LLM-optimized pseudo-code for `/plan` consumption.
Also handles FDD + API Contracts RAG indexing.

## Prerequisites (from detail.md router)
- FDD document exists (generated by detail/fdd.md)
- DESIGN_SCOPE is "frontend" or "fullstack"

---

## Step 1: Load FDD Document

```pseudo
fdd_file = find("documents/features/{feature_dir}/*-frontend-detail-design.md")
srs_file = find("documents/features/{feature_dir}/*-srs.md")
fdd_content = readFileSync(fdd_file, "utf8")
fdd_line_count = fdd_content.split("\n").length
```

---

## Step 2: Generate FDD Pseudo-code

```pseudo
fdd_pseudo = PseudoCodeGenerator(fdd_file, srs_file, "FRONTEND_DD")

# Size-based trigger: markers + > 300 lines -> multi-file
if fdd_content.includes("<!-- DD-SECTION:") AND fdd_line_count > 300:
    fdd_pseudo.setSplitMode("multi-file")
    result = await fdd_pseudo.generateAndSplit()

    # Save master
    masterPath = fdd_file.replace(".md", ".pseudo")
    writeFileSync(masterPath, result.master)
    display(f"FDD pseudo-code (multi-file master): {masterPath}")

    # Save section files
    for [sectionId, content] in result.sections:
        sectionPath = fdd_file.replace(".md", f"-{sectionId}.pseudo")
        writeFileSync(sectionPath, content)
        display(f"  Section: {sectionId}.pseudo")
else:
    # Single file (default)
    pseudo_path = fdd_file.replace(".md", ".pseudo")
    fdd_pseudo.save(pseudo_path)
    display(f"FDD pseudo-code generated: {pseudo_path}")
```

---

## Step 3: Validate Pseudo-code (non-blocking)

```pseudo
report = PseudoCodeValidator(pseudo_path, fdd_file, srs_file)
report.save(CONTEXT_DIR)
# Non-blocking: report but don't stop
```

---

## Step 4: RAG Index FDD + API Contracts (non-blocking)

```pseudo
try:
    rag = HippoRAGService.getInstance(feature, branch)

    # Index FDD
    await rag.indexContent(fdd_content, "fdd", "DD_COMPLETE")
    await rag.extractAndUpdate(fdd_content, "fdd", "frontend-detail-design")
    display("FDD indexed into RAG (design layer)")

    # Index API Contracts if exists
    api_file = find("documents/features/{feature_dir}/*-api-contracts.md")
    if api_file exists:
        api_content = readFileSync(api_file, "utf8")
        await rag.indexContent(api_content, "fdd", "DD_COMPLETE")
        display("API Contracts indexed into RAG (design layer)")
except:
    display("RAG indexing skipped (non-blocking)")
```
