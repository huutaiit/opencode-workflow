# BDD Pseudo-code Generation

## Purpose
Transform BDD document into LLM-optimized pseudo-code for `/plan` consumption.
Also handles BDD RAG indexing.

## Prerequisites (from detail.md router)
- BDD document exists (generated by detail/bdd.md)
- DESIGN_SCOPE is "backend" or "fullstack"

---

## Step 1: Load BDD Document

```pseudo
bdd_file = find("documents/features/{feature_dir}/*-backend-detail-design.md")
srs_file = find("documents/features/{feature_dir}/*-srs.md")
bdd_content = readFileSync(bdd_file, "utf8")
bdd_line_count = bdd_content.split("\n").length
```

---

## Step 2: Generate BDD Pseudo-code

```pseudo
bdd_pseudo = PseudoCodeGenerator(bdd_file, srs_file, "BACKEND_DD")

# Size-based trigger: markers + > 300 lines -> multi-file
if bdd_content.includes("<!-- DD-SECTION:") AND bdd_line_count > 300:
    bdd_pseudo.setSplitMode("multi-file")
    result = await bdd_pseudo.generateAndSplit()

    # Save master
    masterPath = bdd_file.replace(".md", ".pseudo")
    writeFileSync(masterPath, result.master)
    display(f"BDD pseudo-code (multi-file master): {masterPath}")

    # Save section files
    for [sectionId, content] in result.sections:
        sectionPath = bdd_file.replace(".md", f"-{sectionId}.pseudo")
        writeFileSync(sectionPath, content)
        display(f"  Section: {sectionId}.pseudo")
else:
    # Single file (default)
    pseudo_path = bdd_file.replace(".md", ".pseudo")
    bdd_pseudo.save(pseudo_path)
    display(f"BDD pseudo-code generated: {pseudo_path}")
```

---

## Step 3: Validate Pseudo-code (non-blocking)

```pseudo
report = PseudoCodeValidator(pseudo_path, bdd_file, srs_file)
report.save(CONTEXT_DIR)
# Non-blocking: report but don't stop
```

---

## Step 4: RAG Index BDD (non-blocking)

```pseudo
try:
    rag = HippoRAGService.getInstance(feature, branch)
    await rag.indexContent(bdd_content, "bdd", "DD_COMPLETE")
    await rag.extractAndUpdate(bdd_content, "bdd", "backend-detail-design")
    display("BDD indexed into RAG (design layer)")
except:
    display("RAG indexing skipped (non-blocking)")
```
