## Responsibility

Generate Section 2 of Test Plan containing:
- 2.1 Phủ theo tính năng (Coverage by Feature) — Feature | Total | Automated | Manual | Coverage%
- 2.2 Phủ theo ưu tiên (Coverage by Priority) — P0=100%, P1=60%, P2=20%
- 2.3 Truy vết yêu cầu (Requirements Traceability) — FR-ID → Test IDs mapping

## Agent Prompt

### Step 1: PRE-GENERATION REASONING

**THINK**:
1. What FRs exist in SRS? (complete list)
2. What test IDs were generated by tp-03→08? (from section_summaries)
3. Which FRs are covered? Which are gaps?
4. What is the coverage percentage per priority level?
5. Are P0 requirements at 100% coverage?

**REASON**:
- FROM section_summaries → aggregate all test IDs per FR
- FROM SRS → get complete FR list
- CALCULATE coverage: covered_FRs / total_FRs * 100
- IDENTIFY gaps: FRs with zero test coverage
- IF P0 coverage < 100% → flag as CRITICAL gap

**VALIDATE CONSTRAINTS**:
- [ ] Every FR mapped to at least 1 test ID
- [ ] Coverage percentages correct
- [ ] Gap analysis present
- [ ] Vietnamese >= 60%

### Step 3: SELF-CRITIQUE (Q1-Q4)

**Q1**: Every FR from SRS mapped to at least one test ID?
**Q2**: Coverage percentages sum correctly and meet 80%+ target?
**Q3**: Vietnamese >= 60%?
**Q4**: No implementation code?

### Step 4: VALIDATION

- [ ] 4 sub-sections (2.1 Feature, 2.2 Priority, 2.3 Traceability, 2.4 Gap)
- [ ] All FRs from SRS present in traceability table
- [ ] P0 coverage = 100% (or flagged as gap)
- [ ] Total coverage ≥ 80%
- [ ] Gap analysis identifies uncovered FRs
- [ ] section_summaries correctly aggregated
- [ ] ≤ 170 lines
